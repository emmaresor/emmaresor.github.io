    GERP

    Gerp is a verson of the terminal search function grep. Coded in C++, gerp allows a user to search for a keyword in a set of text files in a file tree and returns every instance of the word along with its filepath, text file name, line number it appears on, and the line itself. Includes both case sensitive and insenitive modes. Utilizes an expandable hash table with linear probing (for collision avoidanve) for quick lookup; can hash ____ of stuff in ___ seconds with near-instant queries. Completed November 2016.

    UNBLACKEDGES

    Unblackedges takes an image (pnm) file and removes all black edge pixels by turning them white. Coded in C with my partner Daniel Katz, unblackedges utilizes the data in a pnm file to populate a 2D unboxed bit array, scan the bits, and switch black bits to white if they are on the image border or are the neighboring bit of a black image border pixel. It then prints the remedified file to stdout.
